GitZen

A TUI Git client inspired by lazygit, written in Go with Bubble Tea.

Quick Install

Linux / macOS

curl -sSL https://quanghai2k4.github.io/gitzen/install | bash

Windows (PowerShell)

irm https://quanghai2k4.github.io/gitzen/install.ps1 | iex

Uninstall

gitzen --uninstall
